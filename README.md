## 💻 How to Run Locally
블로그 레이아웃을 로컬 환경에서 테스트하고 변경 사항을 미리 확인하고 싶으시다면 아래 과정대로 실행해 주세요.

```bash
# 1. 저장소를 로컬로 클론합니다.
git clone https://github.com/ji7163/ji7163.github.io.git

# 2. 다운로드된 폴더로 이동합니다.
cd ji7163.github.io

# 3. Ruby 패키지들을 설치합니다. (Bundler 필요)
bundle install

# 4. 로컬 서버를 실행합니다.
bundle exec jekyll serve
```
실행 완료 후 터미널에 뜨는 주소(`http://127.0.0.1:4000/`)로 접속하시면 방금 구성한 블로그를 확인하실 수 있습니다.
